# 🪑 Antiquers Room

## 👀 Overview

- "엔틱 가구 경매 플랫폼"
- 엔틱 가구를 경매로 판매/구매 할 수 있도록 연결해주는 플랫폼 

## 🛠 Tech Stack
- Infra - AWS EC2, RDS, S3
- DB - MariaDB
- Backend - Java, Spring Framework, Mybatis 
- Frontend - jQuery, Bootstrap
